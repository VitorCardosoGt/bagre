===== Stack resumida =====

^ Camada     ^ Tecnologia ^
| Backend    | Node 20 + Fastify + Prisma 5 |
| Banco      | PostgreSQL 16 |
| Frontend   | React 18 + Vite + Tailwind + React Query |
| Proxy/TLS  | nginx (no container ''web'') |
| Orquestração | Docker Compose |

===== Containers em produção =====

^ Container        ^ Imagem               ^ Função ^
| ''bagre-db''  | postgres:16-alpine   | Banco de dados |
| ''bagre-api'' | build local (Node 20) | API REST + scheduler Zabbix |
| ''bagre-web'' | build local (nginx)   | UI estática + proxy ''/api'' |
