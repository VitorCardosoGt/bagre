===== Fluxo: Alocar próximo IP livre =====

<code>
ADMIN             Browser                api                       db
  │                  │                    │                         │
  │  /subnets/5      │                    │                         │
  │  clica em        │                    │                         │
  │  "Alocar próximo │                    │                         │
  │  livre"          │                    │                         │
  ├─────────────────▶│                    │                         │
  │                  │  GET /api/subnets/5/next-free-ip            │
  │                  ├───────────────────▶│                         │
  │                  │                    │  SELECT IpAddress       │
  │                  │                    │  WHERE subnetId=5       │
  │                  │                    │   AND status='FREE'     │
  │                  │                    │  ORDER BY id ASC LIMIT 1│
  │                  │                    ├────────────────────────▶│
  │                  │                    │◀────────────────────────┤
  │                  │  {id:123, address: │                         │
  │                  │  "10.150.0.15", ...│                         │
  │                  │◀───────────────────┤                         │
  │                  │                    │                         │
  │                  │  abre AllocateIpModal com address pré-preenchido
  │                  │                    │                         │
  │                  │  (busca lista de Devices via /api/devices)   │
  │                  │                    │                         │
  │  preenche form   │                    │                         │
  │  (Novo Device:   │                    │                         │
  │  srv-xx, type:   │                    │                         │
  │  Linux, etc)     │                    │                         │
  ├─────────────────▶│                    │                         │
  │                  │  POST /api/ips/123/allocate                 │
  │                  │  { device: {name:"srv-xx",...},             │
  │                  │    hostname:"srv-xx", function, notes }     │
  │                  ├───────────────────▶│                         │
  │                  │                    │  Prisma transaction:    │
  │                  │                    │  - CREATE Device        │
  │                  │                    │  - UPDATE IpAddress     │
  │                  │                    │     deviceId, status,   │
  │                  │                    │     hostname, type, ... │
  │                  │                    │  - INSERT AuditLog      │
  │                  │                    │     entity=ip,          │
  │                  │                    │     action=allocate     │
  │                  │                    ├────────────────────────▶│
  │                  │                    │◀────────────────────────┤
  │                  │  { ip: {...},      │                         │
  │                  │    device: {...} } │                         │
  │                  │◀───────────────────┤                         │
  │                  │                    │                         │
  │                  │  invalidateQueries(['subnet', 5])            │
  │                  │  invalidateQueries(['devices'])              │
  │                  │  → tabela recarrega                          │
  │                  │  → IP aparece como USED                      │
  ▼                  ▼                    ▼                         ▼
</code>

===== Camadas de auth =====

<code>
Request
   │
   ▼
┌──────────────────────────────────────────────────────┐
│  Hook onRequest (apps/api/src/index.js)              │
│                                                       │
│  1. URL em PUBLIC? ─── yes ──▶ deixa passar         │
│         │                                             │
│         no                                            │
│         ▼                                             │
│  2. jwtVerify(req.headers.authorization)             │
│         │  ok                  │  fail               │
│         ▼                      ▼                     │
│  3. SELECT User WHERE          401 unauthorized      │
│     id = jwt.sub                                      │
│     AND active = true                                 │
│         │                                             │
│         ▼                                             │
│  4. método write?                                     │
│     POST/PATCH/PUT/DELETE                            │
│         │                                             │
│         yes                                           │
│         ▼                                             │
│  5. user.role === 'ADMIN'?                           │
│     OR  url === '/api/auth/change-password'          │
│     OR  url === '/api/auth/me'                       │
│         │  yes                 │  no                  │
│         ▼                      ▼                     │
│     deixa passar          403 forbidden              │
│                                                       │
└──────────────────────────────────────────────────────┘
   │
   ▼
Handler da rota (com req.user já populado)
</code>
