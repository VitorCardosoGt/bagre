===== Pendente =====

==== Aguardando ação externa ====

  * **Zabbix sync** — aguardando URL + API token (você vai pegar com o admin do Zabbix). Veja [[tecnico:integracoes#Zabbix|seção Zabbix]].
  * **Fortinet sync** — aguardando URL + API token + liberação de firewall (1 FortiGate em SP3). Veja [[tecnico:integracoes#Fortinet|seção Fortinet]].
  * **Cert TLS válido** (não self-signed) — quando a CA interna Bagre emitir, substituir os 2 arquivos em ''infra/tls/'' e ''docker compose restart web''.
